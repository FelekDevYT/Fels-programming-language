# Official fels programming language change log

## Version symbols

- f: failed version
- t: testing version
- A,B,C,D...: Subversions, usually added in the same day
- a,b,c,d...:build versions

> ### Examples of versions
> - 0.7Aa
> - 0.7Def
> - 0.7
> - 0.7Dt
